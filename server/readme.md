# server files

## Pre-Requirements

- [MongoDB](https://www.mongodb.com/)
- [Node](https://nodejs.org/en/) ^10.0.0
- [yarn](https://yarnpkg.com/)

## Server-Side Usage (PORT: 4000)

```terminal
$ cd server   // go to server folder
$ yarn install   // yarn install pacakges
$ yarn start // run it locally
```
